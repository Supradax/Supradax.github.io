[Data Structure]Stack, Queue and Monotonic Version

March 23, 2023 in Data Structure

栈与队列是两种对操作位置有限制的特殊线性表，栈遵循FILO(First In Last Out)，一般只支持查询栈信息、压栈和弹栈的操作(即只对栈顶操作)；队列遵循FIFO(First In First Out)，一般支持查询队头队尾信息和操作，即在队头出队、队尾入队。

1 栈与队列的实现

栈，一般基于普通顺序表实现，需要满足top(), push(), pop(), empty()方法. 由于无需进行随机访问，可选用链表减少维护成本，其中C++ STL提供了栈的模板类stack<T>

由于栈只从一个方向进行操作且不时有弹栈操作，大多数时候不需要占用到整体长度的一半，故而在一个数组两端可分别建立一个栈，即双向栈。当两个栈栈顶索引相差1时，表示此栈已满。

2 栈与队列的基本操作

2.1 栈的基本应用

Problem A 括号匹配问题

给定一个只包含() [] {}三种括号的字符串str，判断该字符串中的括号是否已经正确匹配. Analysis 只需建立一个栈，当栈顶括号为左括号且当前指针指向的为相应右括号时，正确匹配，故弹栈；否则入栈。若全部正确匹配，则最终结果为空栈

Problem B 逆波兰表达式与符号栈

给定中序表达式，包含加减乘除四种运算，试计算其结果. Analysis 设定数值栈与符号栈，当当前指向符号优先级高于栈顶时，需弹出栈顶两个元素进行运算，否则入栈；此题中只有两个优先级，该法则可适用于含有更多的优先级的表达式，如左括号为0，加减为1，乘除为2，右括号为3

一般地，用栈解决该类问题时无论输入是波兰表达式、逆波兰表达式或是中序表达式，处理大同小异

Problem C 递归（DFS）模拟

LCM问题（最近公共祖先问题）：给定一个二叉树的根节点和两个节点，返回其最近公共祖先节点 Analysis 用栈模拟DFS过程，由于树中任意两个节点间路径有且仅有一条，从而只需比对其路径即可

3 单调栈

在有些时候，我们可能对某种定义下的局部极值感兴趣，在特定的遍历顺序中一些峰值会让其他元素失效(有更高优先级)，此时则需要维护一个单调递增或递减栈。

在使用单调栈时，需要考虑能否使用（具有一定序关系）、单调递增还是单调递减、严格还是非严格，下面是基本模板(数组实现)：

Problem A 每日温度

Problem 给定一个表示每日温度的序列day[i]，返回一个数组next[i]，next[i]表示第i天之后的第一个比第i天高的温度(若一直不升则用0表示)

Input: temperatures = [73,74,75,71,69,72,76,73]
Output: [74,75,76,72,72,76,0,0]

Analysis 注意到序关系——我们只关心第一个温度更高的，且由于顺序遍历时往后的元素存在不可预见性，需要倒序遍历并维护一个自顶向下单调递减栈——若前方有温度更高的，则后方温度低的没有作用，从而该单调栈是合理的

Problem B 柱状图的最大矩形

Problem 给定一个柱状图，求所有矩形构成图形中的最大矩形

Analysis 顺序遍历，注意到当矩形高度递增时，才有可能扩大原有面积；相反，递减时只可能在原有基础上减少（即降低高度上限，只需保存一个最远的且高于当前矩形的横坐标，余下中间的矩形全部没用），通过记录farthest的横坐标防止如2 1 2时最大面积3丢失(即把第一个2的信息丢失)。为避免在程序最后需要把栈倒空，可在首尾添加一个高度为0的矩形.

Tricks

  • 对数组进行边界处理：留空、设置特殊值(如本题设置为0防止栈未被倒空)等
  • 保存索引而不是保存数值本身(否则需要使用pair)，但若需要灵活修改索引，则必须使用pair

Problem C 接雨水

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水

Analysis 本题结合括号匹配和矩形面积的思想，前者即当满足某种序关系时及时更新雨水容量，后者即利用单调栈去除不必要的信息。若当前高度小于栈顶，则意味着该当前与栈顶间可能积水，但是尚不知道后面是否有另一堵墙能拦住，从而暂时入栈，至此可知维护的是自顶向下的单调递减栈。

若当前高度大于等于栈顶，则意味着栈顶与当前之间一定能够储一些水，则开始弹栈并计算此前每一堵比其矮的墙所积水量。类比最大矩形中的farthest，为了避免丢失信息，需要记录下上一堵计算过积水的墙的高度(否则会重复计算)。可以看到在更新单调栈的过程中积水的计算是横向计算而不是纵向计算的。

4 单调队列

和单调栈的唯一区别在于单调队列需要同时从两端维护整体单调性，其余思想与单调栈相同。

Problem 滑动窗口的最大值

Problem 给定滑动窗口长度k，每次该窗口向右滑动一个单位，求第i个窗口内的最大值res[i]

Analysis 注意到序关系，即若已有一个更大元素入队，则此前所有比其小的元素全部失效，从而可以维护一个单调双端队列。其中在出队时只有队头等于上一次最大值时才进行出队
