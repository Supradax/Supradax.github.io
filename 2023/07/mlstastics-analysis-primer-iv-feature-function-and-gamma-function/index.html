[ML]Stastics Analysis Primer IV: Feature Function and Gamma Function

July 23, 2023 in ML, Math

1. 矩母函数

注意到$e^x=\sum\frac{x^n}{n!}$. 两边取期望则有$E[e^{tX}]=\sum{\frac{1}{n!}E\left[t^nX^n\right]}=\sum{\frac{t^n}{n!}E\left[X^n\right]}$. 故对$E\left[e^{tX}\right]$求n次导并令t=0，可立得X的n阶矩而不必进行繁复的多项式展开和代入.

因此定义函数$M_{X}(t)=E[e^{tX}]$$(t\geq0)$是随机变量X的矩母函数（Moment Generation Function：MGF）.根据定义，有$\frac{d^n}{dX}E\left[e^{tX}\right]|_{t=0}=E\left[X^n\right] M_X\left(0\right)=1$，$M_X(t)$连续. 直接验证有如下结论：

命题1 若X和Y独立，则$M_{X+Y}=E\left[e^{t(X+Y)}\right]=E\left[e^{tX}\right]E\left[e^{tY}\right]=M_X\left(t\right)M_Y(t)$

  • 推论：若$X_{1:n}$独立，则$M_{\sum X_i}=\prod{M_{X_i}\left(t\right)}$

命题2 $M_{aX+b}\left(t\right)=E\left[e^{t\left(aX+b\right)}\right]=e^{bt}E\left[e^{(at)X}\right]=e^{bt}M_X\left(at\right)\ $

此外，还有如下定理：

命题3（Bernstein定理） 若$f\left(t\right)(t\geq0)$连续且$f\left(0\right)=1$，则f(t)是一个概率测度的矩母函数 当且仅当 f(t)单调.

Notes. 随机变量X的分布→矩母函数不一定是单射.只有当已知X的分布类型或X取值离散时，M_X(t)才能唯一确定分布（各阶矩提供可列个方程，而某一分布类型的参数是有限的）. 只有特征函数才能唯一确定分布. 一个反例为：.

命题4 常见分布的矩母函数：

  1. $X\sim\exp{\left(\lambda\right)},M_X(t)=\frac{\lambda}{\lambda-t}$
  2. 若X~B(n,p)，则$M_X\left(t\right)=\left(pe^X+1-p\right)^n$
  3. X~U(a,b)，则$M_X(t)=\frac{1}{b-a}\frac{e^{bt}-e^{at}}{t}$
  4. $X\sim N(\mu,\sigma^2)$，则$M_X\left(t\right)=\exp{\mu t+\frac{1}{2}\sigma^2t^2}$
  5. $X\sim P\left(\lambda\right)$，则$M_X\left(t\right)=\exp{\lambda\left(e^t-1\right)}$
  6. $X\sim G(p)$，则$M_X\left(t\right)=\frac{pe^t}{1-\left(1-p\right)e^t}$

pf.

  1. $E\left[e^{tX}\right]=\lambda\int_{0}^{\infty}{e^{-\lambda x}e^{tx}\ }dx=\lambda\int_{0}^{\infty}{e^{x(t-\lambda)}\ }dx=\frac{\lambda}{\lambda-t}$
  2. 考察0-1分布，则$M_X\left(t\right)=pe^X+\left(1-p\right)e^{0X}=pe^X+1-p$.由于B(n,p)中$X=X_1+…+X_n$，其中$X_i$ i.i.d. 故$M_X\left(t\right)=\left(pe^X+1-p\right)^n$
  3. $E\left[e^{tX}\right]=\int_{a}^{b}{\frac{1}{b-a}e^{tx}\ }dx=\frac{1}{b-a}\frac{e^{bt}-e^{at}}{t}$
  4. $E\left[e^{tX}\right]=\int_{-\infty}^{+\infty}{\frac{1}{\sqrt{2\pi}\sigma}e^{tx-\frac{1}{2}\left(x-\mu\right)^2}}dx=e^{-\frac{\mu^2-\left(\mu+\sigma^2t\right)^2}{2\sigma^2}}\frac{1}{\sqrt{2\pi}\sigma}\int_{-\infty}^{+\infty}e^{-\frac{1}{2\sigma^2}\left(x-\mu-\sigma t\right)^2}dx$$=\exp{-\frac{\mu^2-\left(\mu+\sigma^2t\right)^2}{2\sigma^2}}=\exp{\mu t+\frac{1}{2}\sigma^2t^2}$
  5. $E\left[e^{tX}\right]=\sum_{n=0}{\frac{\lambda^n}{n!}\ e^{-\lambda}e^{tn}}=e^{-\lambda}\sum_{n=0}{\frac{{(\lambda e^t)}^n\ }{n!}\ }=\exp{\lambda\left(e^t-1\right)}$
  6. $E\left[e^{tX}\right]=\sum_{n=1}{pq^{n-1}e^{tn}}=\frac{p}{q}\sum_{n=1}\left(qe^t\right)^n=p·qetq(1-qet)=pet1-1-pet$

2. 特征函数

将矩母函数推广到复数域，有$f_X\left(t\right)=E\left[e^{iXt}\right]$. 类比矩母函数，有$f_X\left(0\right)=1$，由于$\left|e^{iXt}\right|=1$，从而$\left|f_X\left(t\right)\right|\le1$ . $f_{aX+b}\left(t\right)=e^{itb}f_X(at)$，在R上一致连续 容易验证$f_X\left(-t\right)=\bar{f_X\left(t\right)}\ $.

命题5 特征函数与分布唯一确定.

引理1·Fubini定理 A,B是σ-有限测度空间，若f(x,y)在A×B的积空间上绝对可积且积分值有上界$(\int_{A\times B}{\left|f\left(x,y\right)\right|d(x,y)}<\infty)$，则可交换累次积分且积分值相等

引理2·Diriclet积分 $\int_{0}^{+\infty}\frac{\sin{t}}{t}dt=\frac{\pi}{2}$

引理3 $D\left(k\right)=\frac{1}{\pi}\int_{0}^{+\infty}\frac{\sin{kt}}{t}dt=\frac{1}{2}\ sgn(k)$

引理4 $x_1<x_2$，$g\left(T,x,x_1,x_2\right)=\frac{1}{\pi}\int_{0}^{T}\left[\frac{\sin{t(x-x_1)}}{t}-\frac{\sin{t(x-x_2)}}{t}\right]dt$，则有$T\rightarrow\infty$时，当$x\in\left(x_1,x_2\right)$时为1，为端点时为$\frac{1}{2}$，否则为0.

引理5·逆转定理 设CDF和特征函数分别为$F\left(x\right),f(t)$.有：$I_T=\frac{1}{2\pi}\int_{-T}^{T}\frac{e^{-itx_1}-e^{-itx_2}}{it}f(t)dt=\frac{1}{2\pi}\int_{-T}^{T}{\frac{e^{-itx_1}-e^{-itx_2}}{it}\int_{-\infty}^{+\infty}{e^{itx}dF(x)}}dt$

引理6·Lesbegue控制收敛定理 测度空间中$f_n$是实值可测函数列，若$f_n$逐点收敛到函数f，且存在一个Lesbegue可积的g，使得$\left|f_n\right|\le\left|g\right|$恒成立，那么：f也是Lesbegue可积且积分时可交换积分算子与极限算子$n\rightarrow\infty$.

pf. 对引理1和6不加以证明.

Lemma 2. 引入积分收敛因子$e^{-th}$进行求解.$\int_{0}^{+\infty}\frac{\sin{t}}{t}dt=\int_{0}^{+\infty}{\int_{0}^{+\infty}{e^{-th}\sin{t}}dtdh}=-\int_{0}^{+\infty}{\frac{1}{1+h^2}\left(h\sin{te^{-th}}+\cos{te^{-th}}\right)\mid_0^\infty}dh=\int_{0}^{+\infty}\frac{1}{1+h^2}dh=\frac{\pi}{2}$. 其中第二个等号t分部积分2次积分再现，可交换性由(第二个等号利用$e^x\geq x^2$)$\int_{0}^{+\infty}\int_{0}^{+\infty}\left|e^{-th}\sin{t}\right|dtdh\le\int_{0}^{+\infty}{\int_{0}^{+\infty}e^{-th}dtdh}\le\int_{0}^{+\infty}{\int_{0}^{+\infty}\frac{1}{t^2h^2}dtdh}$保证.

另一种方法为积分号下求导. $I\left(a\right)=\int_{0}^{+\infty}\frac{\sin{t}}{t}e^{-at}dt$，则$I^\prime\left(a\right)=\frac{d}{da}\int_{0}^{+\infty}\frac{\sin{t}}{t}e^{-at}dt=-\int_{0}^{+\infty}\sin{t}e^{-at}dt$，积分再现解得： $I^\prime\left(a\right)=-\frac{1}{1+a^2}$. 从而$I\left(a\right)=-\tan^{-1}{a}+\frac{\pi}{2}$. 故$a\rightarrow0$时，$I\left(a\right)=\frac{\pi}{2}$. 偏导的交换由导函数$I^\prime\left(a\right)$连续保证.

Lemma 3. $D\left(k\right)=\frac{1}{\pi}\int_{0}^{+\infty}\frac{\sin{kt}}{t}dt=\frac{1}{\pi}\int_{0}^{+\infty}\frac{\sin{kt}}{kt}d\left(kt\right)(k>0)$，k<0时积分上下限交换；k=0显然为0.

Lemma 4 显然极限情况下，$g=D\left(x-x_1\right)-D(x-x_2)$，分类讨论即可.

Lemma 5 由Fubini定理，$\int_{-T}^{T}\int_{-\infty}^{+\infty}{\left|\frac{e^{itx}}{it}\right||e^{-itx_1}-e^{-itx_2}|dF(x)}dt\le2\int_{-T}^{T}\int_{-\infty}^{+\infty}{\frac{1}{t}dF\left(x\right)}dt=2\int_{-T}^{T}\frac{1}{t}dt=2\ln{2}$，从而交换积分次序，有$\frac{1}{2\pi}\int_{-\infty}^{+\infty}\int_{-T}^{T}{\frac{e^{-it{(x-x}_1)}-e^{-it(x-x_2)}}{it}e^{itx}dF(x)}dt$

$=\frac{1}{2\pi}\int_{-\infty}^{+\infty}\int_{-T}^{T}{\frac{\sin{t\left(x-x_1\right)}-\sin{t\left(x-x_2\right)}}{t}}dF(x)dt$.

由Euler公式和偶倍奇零（由欧拉公式展开，对称区间积分只剩余弦分类分量(另一部分可直接令$T\rightarrow\infty$而为0），接下来证明极限$T\rightarrow\infty$与积分的可交换性，由Lesbegue控制收敛定理，只需考虑$\left|\int_{n}^{+\infty}\int_{-\infty}^{+\infty}{\frac{\sin{t\left(x-x_1\right)}-\sin{t\left(x-x_2\right)}}{t}dF(x)dt}\right|$.

由于$x_1,x_2$给定，分别考察$\int_{n}^{+\infty}\int_{-\infty}^{+\infty}{\frac{\sin{t\left(x-x_1\right)}}{t}dF(x)dt}$，这与考察$\int_{n}^{+\infty}\int_{-\infty}^{+\infty}{\frac{\sin{tx}}{t}dF(x)dt}$是等价的（阿基米德性质），再交换回去，可得$\int_{-\infty}^{+\infty}\int_{n}^{+\infty}{\frac{\sin{tx}}{tx}d\left(xt\right)dF(x)}$，从而只需证$\int_{0}^{+\infty}\frac{\sin{x}}{x}dx$收敛的，按区间分解掉两两合并即可. 从而给定$\frac{1}{m}$，一定有n>N时，$\int_{-\infty}^{+\infty}\int_{n}^{+\infty}{\frac{\sin{tx}}{tx}d\left(xt\right)dF(x)}<\int_{-\infty}^{+\infty}{\frac{1}{m}dF(x)}=\frac{1}{m}$.

综上，直接把T放到积分号内， $\int_{-∞}^{+∞}g·dF(x)=\int_{R-\left[x_1,x_2\right]\ }{0dF(x)}+\int_{x_1,x_2}{\frac{1}{2}dF(x)}+\int_{\left(x_1,x_2\right)}{dF(x)}=F\left(x_2\right)-F(x_1)$, 令$x_1\rightarrow-\infty， x_2=x$，x为CDF的连续点上，特征函数绝对可积，那么逆转公式唯一确定了F(x). CDF几乎处处连续可导，则两边求导，则有$\rho\left(x\right)=\frac{1}{2\pi}\int_{-\infty}^{+\infty}{e^{-itx}f(t)dt}$. 即PDF和特征函数是Fourier变换关系.

常见分布的特征函数只需将MGF中的t改为it即可得.

3. Gamma函数(第一类Euler积分)

Euler想知道，能否将阶乘从自然数推广至全体实数，进而找到经过(n,n!)的函数. $$\Gamma\left(z\right)=\int_{0}^{+\infty}{x^{z-1}e^{-x}dx}$$

命题5 $\Gamma\left(z\right)$的性质

  1. $\Gamma\left(n\right)=n\Gamma(n-1),\Gamma\left(n\right)=(n-1)!$
  2. $\Gamma\left(\frac{1}{2}\right)=\sqrt\pi$
  3. $\Gamma\left(z\right)$单调递增(z>0).
  4. Gamma分布的PDF积分为1.

pf.

  1. $\Gamma(0)=1,\Gamma(n)=\int_{0}^{+\infty}{x^ne^{-x}dx}=e^{-x}x^n\mid_{+\infty}^0+n\int_{0}^{+\infty}{x^{n-1}e^{-x}dx}=n\Gamma(n-1)$. 利用递推式可推广出$\Gamma\left(n\right)=\infty$，n为负整数
  2. $\Gamma\left(\frac{1}{2}\right)=\int_{0}^{+\infty}{\frac{1}{\sqrt x}\ e^{-x}dx}=\int_{0}^{+\infty}{e^{-t^2}dt}(t=\sqrt x)$. 而$\int_{0}^{+\infty}{e^{-t^2}dt}\int_{0}^{+\infty}{e^{-s^2}dt}=2\pi\int_{0}^{+\infty}{{re}^{-r^2}dr}=\pi$ .从而$\Gamma(\frac{1}{2})=\sqrt\pi$.结合性质1得： $\Gamma(-\frac{1}{2})=-2\sqrt\pi$.
  3. 偏导连续，故可交换求导和积分次序.
  4. 由$\Gamma\left(z\right)=\int_{0}^{+\infty}{x^{z-1}e^{-x}dx}=\int_{0}^{+\infty}{\lambda^{z-1}x^{z-1}e^{-\lambda x}d(\lambda x)}$，故$1=\int_{0}^{+\infty}{{\frac{\lambda^z}{\Gamma\left(z\right)}x}^{z-1}e^{-\lambda x}dx}$，从而PDF为$f\left(z\right)={\frac{\lambda^z}{\Gamma\left(z\right)}x}^{z-1}e^{-\lambda x}$.

4. Beta函数(第二类Euler积分)

对B(n,p)，$P\left(X=k\right)=C_n^kp^k\left(1-p\right)^{n-k}=\frac{n!}{\left(n-k\right)!k!}p^k\left(1-p\right)^{n-k}$. 现在问题从在n次实验中，给定成功概率p，成功k次的概率，变成 在n次实验中，成功k次，成功概率p为$p_0$的概率.那么该分布至少满足$C\int_{0}^{1}{p^k\left(1-p\right)^{n-k}dp\ }=1$，为研究该分布的形式，记：$$B\left(a+1,b+1\right)=\int_{0}^{1}{x^a\left(1-x\right)^b}dx$$

令$x=\sin^2{\theta}$，得Beta函数的三角形式.$$B\left(a+1,b+1\right)=\int_{0}^{\frac{\pi}{2}}{\sin^{2a}{\theta}\cos^{2b}{\theta}}d\ sin^2\theta=2\int_{0}^{\frac{\pi}{2}}{\sin^{2a+1}{\theta}\cos^{2b+1}{\theta}}d\theta$$

命题6 Beta函数的性质.

  1. $B\left(a,b\right)=B(b,a)$
  2. $B\left(a+1,b+1\right)=\frac{b}{a+1}B(a+2,b)\ $
  3. $B\left(a,b\right)=\frac{\Gamma\left(a\right)\Gamma\left(b\right)}{\Gamma\left(a+b\right)}$

pf.

  1. 作变换$x^\prime=1-x$
  2. $B\left(a+1,b+1\right)=\frac{1}{a+1}\int_{0}^{1}\left(1-x\right)^bdx^{a+1}=\frac{b}{a+1}\int_{0}^{1}{x^{a+1}\left(1-x\right)}^{b-1}dx=\frac{b}{a+1}B(a+2,b)$
  3. $\Gamma\left(z\right)=\int_{0}^{+\infty}{x^{z-1}e^{-x}dx}=2\int_{0}^{+\infty}{s^{2z-1}e^{-s^2}ds}$, 换元便于极坐标分离，注意$x,y>0$，故$\theta\in[0,{\pi\over2}]$ $$\begin{align} \Gamma\left(a\right)\Gamma\left(b\right)&=4\int_{0}^{+\infty}{x^{2a-1}e^{-x}dx}\int_{0}^{+\infty}{y^{2b-1}e^{-y}dy}=4\iint{x^{2a-1}y^{2b-1}e^{-(x^2+y^2)}d\sigma} \\&=4\iint{r^{2a+2b-1}\ \cos^{2a-1}\theta\sin^{2b-1}{\theta}\ e^{-r^2}drd\theta}\ \\&=4\int_{0}^{+\infty}{r^{2\left(a+b\right)-1}e^{-r^2}dr}\ \int_{0}^{\frac{\pi}{2}}{\cos^{2a-1}\theta\sin^{2b-1}{\theta}}d\theta\ \\&=2\Gamma\left(a+b\right)\frac{1}{2}B(a,b) \end{align}$$
