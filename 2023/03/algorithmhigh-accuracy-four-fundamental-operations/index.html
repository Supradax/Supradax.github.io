[Algorithm]High Accuracy: Four Fundamental Operations

March 19, 2023 in Algorithm, Simulation

High Accuracy, is adopted when the data structure cannot bear the too large data. We are going to talk about how to realize addition, subtraction, multiplication and division with data structure string.

Among these four operations, a trick widely used is to reverse the string which make it easier to simulate the carry process.

§I Addition

For addition, we simply reverse stirng a and b, record the maximum possible length, add each two bits up and carry if necessary. In the end, we throw away the useless zeros in the front(Remeber to leave a zero when the answer is just 0) and reverse it back.

§II Subtraction

For subtraction, when compared with addition, we need to take care of the appreance of negative numbers which requires a comparition ahead of operation. This would help us determine the sign it ought to be.

Likewise, we define the maxmium possible length as the size of the larger string. And the left part keeps the same as addition.

§III Multiplication

Almost the same as addition but it requires a dual loop thus making its time complexity O(n²). Fortunately, this can be improved to be O(nlogn) via FFT algorithm, which is scheduled to be brought in another essay. Now the possible maximum length is x.size()+y.size()

§IV Division

To simplify the problem, we assume that x/y where x is a big number while y can be contained in an int. But this time, we no longer need to reverse the string at the very beginning which is due to the fact that we calculate the quotient from the front to the back.
