[Algorithm]Scanning Line

March 25, 2023 in Algorithm

Scanning Line Algorithm，扫描线算法，在与端点有关的问题中广泛应用，如一维的区间扫描和二维的矩形。这些问题的共性在于问题的求解只和端点处的状态改变有关，其中在一维扫描线算法中，我们将涉及一个“区间问题”的专题。

1. 1D Scanning Line Problem

Problem A 给定工作时间的无交区间列并进行合并，求合并后长度有限的补闭区间（空闲时间）

下面是扫描线模板：一般需要将点的类型区分（开始点还是结束点）并进行排序，确定遍历顺序后逐个处理

Problem B 区间系列问题 [常用算法] 线段树、扫描线、双指针、贪心、动态规划

  1. 给定闭区间列，删掉最少的区间使其无交

Solution 本题即活动安排问题（给定活动时间列，在一天内的同一个地点安排最多活动）的反面。可使用贪心或线段树，不宜用扫描线(活动数目和端点没有直接联系)。贪心思想则为越早结束越好，故排序右端点再进行顺序遍历（也可排序左端点进行逆序遍历，转换为越晚开始越好）

  1. 给定闭区间列，合并有交的区间 Solution 本题即Problem A的反面；也可使用线段树(区间段+1)加快慢指针检索非零部分；还可使用贪心思想：能够合并的一定相邻 ，从而对左端点排序进行遍历，判断是否可以合并，直至不能再合并则保存

3.给定无交闭区间列，插入一个新区间，若有有交的，则合并 4.给定无交左闭右开区间列，给定一个区间，删除与之有交的部分 Solution 扫描线. 比对区间端点，记录最左和最右点 5.给定区间列表，删除被包含的区间. Solution贪心. 对左端点排序，记录区间右端点的最大值. 往后区间右端点小于该最大值的则代表有一个区间I 覆盖该区间(I的左端点一定在该区间左侧) 变式1给定会议时间列，判断一个人能否参加全部会议. (将(5)检测右端点小于max改为左端点) 变式2给定会议时间列，判断开满所有会议所需最少会议室. (同类：最少座位数) [转换] 给定区间列，判断最大重叠部分的区间数. 使用扫描线. 入点cnt+1,出点cnt-1，返回max

6.给定区间列，选择一些数使每个区间都至少含有一个数，返回所需的最小数目 Solution 贪心. 对左端点排序，右端点升序. 则“尽可能右移到当前未被标记的区间的最右点”从而使这个数 落到往后的更多区间内，直至下一个区间左端点超过当前最右点.

7.给定闭区间列C和区间I，要求从C中选取最少的区间覆盖I，并返回所需最小数目.(保证一定存在覆盖) Solution ①扫描线. 统计目标区间内每个端点的覆盖区间数，取最小值. ②贪心.按左端点排序，选取区间[i,j]，枚举并选择此后所有能覆盖i的右端点最大的区间.

2. Scanning Line in 2D Problem

Problem A 给定矩形簇左下角和右上角坐标，求：①判断矩形是否恰好无交地构成一个大矩形；②矩形覆盖的总面积；

Problem B: SkyLine 给出矩形建筑的坐标区间和高度，返回其轮廓线的转折点.

Problem C 实现一个类，满足：①将一个区间标记；②将一个区间取消标记；③判断区间是否被整体标记
