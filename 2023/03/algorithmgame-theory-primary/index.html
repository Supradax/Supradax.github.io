[Algorithm]Game Theory: Primary

March 25, 2023 in Algorithm, Game

Game theory, 博弈论，不同于经济学意义上追求共同利益最大的策略，本文更多研究通过最优策略在某个游戏中的零和博弈，即公平组合游戏(ICG：Impartral Combinatorial Game)。常见的形式是，给定一个游戏规则和一个初始状态，在双方对垒中，每个人都使用最优策略进行操作，则游戏结果由规则和初态唯一确定。

显然诸如斗地主的游戏不在本文讨论范围内，因为该类游戏信息不对称，如对手并不知道你的具体手牌(有一部分状态未知)。在有状态未知的情况下，只能采取相对最优策略，没有所谓必胜或必败的概念

若将各个状态表示为节点，则每一次合法操作均会将一个状态转移到另外一个状态，那么将这两个状态用边连接，从而得到一个有向图G。由于初态是给定的且最终目标状态(获胜状态)是唯一的，从而G是单源单汇有向图，即有且仅有一个出度为0和一个入度为0的顶点。

一般地，在假定两个人都用最优策略的情况下，不会出现几个状态反复出现(这不会帮助他们获胜)(也不排除有些时候状态均不可逆)，从而可以规定G中无环。进而得到一个单源单汇的DAG(Directed Acyclic Graph).

在很多情况下，我们研究Nim问题（回合制取物游戏），一般规定取走最后一个物品的人获胜或者轮到某人回合且该玩家无法做出合法操作为负，可以看到两种定义是等价的，而无法做出任何操作这个状态正是游戏的唯一最终态，也即为上述的汇点。

对于该类问题，如果两个人均使用最优策略，那么游戏结果将由初态唯一确定，也即无论怎么操作，先手要么必胜，要么必败，这和具体的游戏过程无关(当然若有人不够bright，那么或许在某个节点他可能将把必胜状态拱手让人，这样对方一直采取最优策略，就能达到转败为胜)

为了更好解决这类问题，我们通过下述例题引出必败点与必胜点的概念.

1 单堆Nim：Bash Game

Problem ALice和Bob在玩取物游戏，规则是每次可以从n个石子里拿走1~3个，拿走最后一个石子的人获胜。若Alice先手且两个人均使用最优策略，对给定的n，Alice是必败还是必胜？

Analysis 我们先确定游戏终点，显然当剩余石子为0时游戏结束。那么是否能够确定一些n使得先手必败呢?

答案是显然的。当n=0时，Alice一上来就没有石子可以拿，属于必败；而n=1,2,3时Alice可以选择一把全拿走，这些状态Alice必胜，那么为什么在n=3时Alice不一次拿1个或2个，非要一次全拿走呢? 读者不难想到，这样使石子剩下1个或2个，轮到Bob时他全拿走，那么Alice必为负。将这一过程抽象出来——这种匪夷所思的操作，会让Bob进入必胜态(n=1,2)，从而导致Alice必败.

那么接下来看n=4，此时无论Alice拿1,2,3个，均会导致Bob接手时剩余1,2,3个，即均为Bob的必胜态。即n=4时Alice无法通过一次操作把棋局变为必败态送给Bob，或者反过来说，Alice不论怎么操作都会让Bob必胜。

因此可得这样的结论：

  • 在零和博弈中，Alice必胜则Bob必败；反之亦然
  • 当无法通过一次合法操作将当前状态S变为对手的必败态时，S为必败态 (也可表述为：不论如何操作S为S’，S’到对手手上都是必胜态)
  • 当存在一个合法操作能将当前状态S变为对手的必败态时，S为必胜态 注意这里的表述，必败的要求是任意操作都不能到必败态，必胜的要求则简单得多，只需要存在一种操作转成必败态，这体现了人的主观能动性(即一直采取能赢的最优策略)

如下图，DAG中蓝色、红色节点表示必胜态、必败态，箭头表示通过合法操作后的状态转移方向，只有蓝色箭头代表的操作能够将必败态送给对方，从而使自己获胜。其中1,2,3,5均有一个蓝色箭头，从而为必胜态；4则没有，从而为必败态。

不难归纳假设，n为4的倍数时先手必败，其余均为先手必胜。从而对Alice而言，她的获胜期望为3/4.

我们可作如下推广，若一次可以取1~m个石子，则m|n时先手必败，反之必胜。先手获胜期望为1-1/m.

3 SG函数

现在将Bash Game中的问题进一步抽象以便于解决情况更为复杂的问题： 我们定义mex(X)函数(minimum excludant)，其中X是一个非负整数集合，对应函数值是X中未出现的最小非负整数 现在用SG(X)来量化必胜态和必败态，即我们判断的状态不再是0,1两种可能，而是包含更多信息

我们定义，SG(X)=mex({SG(X’)})，其中X’为X通过一次操作能到达的状态，以Bash Game为例，定义n=k为状态k，从而

sg(0)=0, sg(1)=mex({sg(0)})=mex({0})=1, sg(2)=mex({sg(0),sg(1)})=2

sg(3)=mex({0,1,2})=3, sg(4)=mex({1,2,3})=0

容易发现，若定义ICG游戏终点对应状态为0，则有：

  • sg(x)=0时，该状态为先手必败态
  • sg(x)≠0时，该状态为先手必胜态

下面证明上述结论：

  1. sg(x)=0，即没有一个操作能够将其转换为必败态给对方
  2. sg(x)≠0，由sg函数定义，必存在一个操作能将x转为必败态，成立

4 Nim Game与SG定理

现在将Bash Game进一步复杂化，得到Nim Game. Problem 现在给定n堆石子，每一堆的石子数目也给定。若现在Alice和Bob每次需要在一堆石子中拿石子，数量没有限制(但一定要拿)，拿走最后一个石子的人获胜，若Alice先手且两个人均使用最优策略，对给定的n，Alice是必败还是必胜？

可以发现，该问题实质上是组合博弈问题，即

  • 一整个游戏可以分为多个子游戏(存在子结构)
  • 每堆石子之间互不干扰(子结构相互独立)
  • 整个游戏结束当且仅当所有子游戏结束
  • 每个回合玩家只能操作一个子游戏

从而可应用SG定理，即 群(X,∪)和群(SG(X),⊕)构成群同态 (⊕为二进制的异或) $$sg(A_1 \cup A_2 … \cup A_n)=sg(A_1)⊕sg(A_2)⊕…⊕sg(A_n)$$

对SG定理，需要用群论的知识——博弈群进行严格证明，放在文末.

也就是说，我们只需将其进行拆分，考虑每一堆石子，显然sg(k)=k(k为任意正整数) 从而整体的sg值为各堆石子数异或的结果，只有不为0时Alice才必败，我们看下不严谨的证明得到一个感性认识：

  • 整体异或为零，则任何操作都会导致整体异或非零 (这和必败态不可能通过操作把必败态给对方同个意思)
  • 整体异或非零，则一定存在一个操作使得整体异或为零 (这和必胜态一定能把必败态送给对方同个意思)

[prerequisite] 异或的性质(相同为0，相异为1)：1^1=0^0=0，0^1=1^0=1且满足交换律和结合律；n个数异或后的结果的第k位为1，则这n个数中一定有奇数个数在第k位为1.

设操作前为ai，操作后为b

  1. 证明1：注意我们每一次只能改变其中一个状态(不能同时操作多堆石子)，从而 由于状态无环，从而前后状态一定不等，反证：假设异或后整体为零 $$a_1⊕…⊕a_i…⊕a_n=0 \Rightarrow a_i=0⊕a_i=(a_1⊕…⊕b⊕…a_n)⊕a_i=0⊕b=b$$ $$\Rightarrow a_i=b \Rightarrow\Leftarrow!$$

  2. 证明2：假设整体异或结果s中二进制最高位为第k位，那么所有状态里面一定存在奇数个第k位为1，从中选一个有t个石子的堆，一定可从石子堆中取m个(m<t)使得当t换为m时整体异或（mex函数的定义）

Problem A Another Nim Game

Problem 假设有n堆石子，每堆当中有若干个石子。两人轮流取石子，可以从任意一堆石子当中取走若干个石子，或者是选择一堆大于1颗的石堆将拆分成两堆。最后无法取走石子的人落败。问：对给定的n个石堆，先手的胜负.

Analysis 本题仍可将n堆石子当成n个独立子游戏，而带来麻烦的石堆拆分实际上可以相当于从一个子游戏分裂成2个子游戏，仍可套用SG定理，即设当前石子有n个，则sg(i+j)=sg(i)^s(j), 其中i+j=n.

Problem B 棋盘游戏II

Problem 给定一个m×n棋盘，每一行均放有一个黑子和白子(初始放置位置已知)，若黑子先手，且每次操作必须且仅能横向移动一个棋子且不能跨越另外一个棋子，当有一名玩家轮到且无法移动任何一枚棋子时为负。问：对给定的初始摆放位置，黑子的胜负.

Analysis 确定终止状态，当所有黑子和白子紧靠一起时，先手必败. 因为后手可以跟着先手一起动，直至将其逼至墙角. 所以等价于有n堆石子（这里石子表示每一行的独立子游戏），每次只能从中取1~n个(等价于移动距离)
