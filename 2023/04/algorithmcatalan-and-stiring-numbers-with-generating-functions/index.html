[Algorithm]Catalan and Stiring Numbers with Generating Functions

April 25, 2023 in Simulation

In combinatorial math, Catalan Number is a sequence that occur in lots of counting problems which are more or less related to the line chart.

1 Counting in a Line Chart

Theo. 1 With a grid and two points$P(0,b_0),Q(n,b_n)$ given, if allowed to draw a diagonal so as to match up P and Q, this is possible iff: $$|b_n-b_0|\leq n, 2|(|b_n-b_0|+n) \tag{1}$$

pf.

  1. Necessity. $|b_n-b_0|\leq \sum{b_n-b_{n-1}}=n$ and decompose each diagonal as $i+j, i-j$ and thus $k+|b_k-b_0|$ adds 2 or remains the same as k adds 1, making it is always even.
  2. Sufficiency. We may suppose it begins at $(0,0)$ and ends at $(n,m)$, and cut off a right triangle at the right corner and fill the rest in the left with a shape “/\” . This is possible because n-m is even. Therefore a construction proof is given.

Theo. 2 The total number of different lines is $\binom{n}{\frac{n+b_n-b_0}{2}}$

pf. There is n segments in total including $x$ “/” and $y$ “\” s.t. $n=x+y,x-y=b_n-b_0$. It implies that $x=\frac{n+b_n-b_0}{2}$

Problem A If the polyline has no intersection with the x-axis and how many feasible methods are there?

  Sol. If there is an illegal line,say it intersects with the x-axis, we can flip the part on the left of the first intersection along the x-axis. It is obviously such symmetry transformation is a bijection which means it is equivalent to know how many different lines between $P’(0,-b_0),Q’(n,b_n)$ are there. It indicates that $\binom{n}{\frac{n-b_n+b_0}{2}}$

Problem B What if the polyline is allowed to have intersections but not to transcend x-axis?

  Sol. It is equivalent to not allowing it to intersect with $y=-1$ and therefore this time we flip along $y=-1$.

2 Catalan Number

Problem If the order of elements pushed into the stack is confirmed, how many possible pop-out sequence are there?

Sol. Let it be $C(n)$. It is easy to divide the problem into many, say, we can label the last element and check the stack size when it is pushed in. If there are k elements inside, it means there are C(k) methods to go on and C(n-k) methods for the previous n-k elements to come along with. Thus obtaining, $$C(n)=\sum_{i=0}^{n-1}C(i)C(n-i),C(0)=1 \tag{2.1}$$

Notice that (2.1) is somewhat similar to Cauchy Product, $$f(x)=1+c_1x+…, f^2(x)=\sum_{k=0}^{\infty}\sum_{i+j=k}c_ic_jx^k=\sum_{k=0}^\infty c_{n+1}x^n$$, thus,

$$xf^2(x)=f(x)-c_0=f(x)-1 \tag{2.2}$$

Figure it out we have $f(x)=\frac{1-\sqrt{1-4x}}{2x}$ and expand it with Taylor series, $$f(x)=\sum_{n=0}^\infty\frac{C_{2n}^n}{n+1}x^n, C(n)=\frac{C_{2n}^n}{n+1}\tag{2.3}$$

Therefore, to verify whether the solution follows Catalan Number, we can test it with (2.1) or (2.3)

There are many scenarios where Catalan Number hides behind

  • the numbers of methods from A to B in a grid along the diagnals(2.1);
  • the methods to divide a polygan into n parts(choose 2n points on a circle and match them up to get n segments without intersections)(2.1);
  • the amount of BST with n nodes given(2.1)
  • with n pairs of brackets, the amount of strings where the brackets match up correctly(the same as the stack problem)(2.1)

3 Generating Function

3.1 Generating Function

There is a type of polynomials whose value is not important but their indices and coefficients matter in the counting.

To work out $a_n$ s.t. $a_0=1,a_1=3,2a_{n+1}=3a_n-a_{n-1}$, let $f(x)=a_0+a_1x+a_2x^2+…$, then we have $(2x^2-3x+1)f(x)=1$, and $f(x)=\frac{1}{2x^2-3x+1}=\frac{1}{x-1}-\frac{2}{2x-1}$ $=-(1+x+…)+2(1+2x^2+4x^3)=\sum(2^{k+1}-1)x^k$ i.e. $a_n=2^{n+1}-1$

The calculation of generating function requires methods like changing the order among infinite series, derivative and integral and applying Taylor expansion.

Problem A(Composition in the multi-sets and the number of solutions to uncertain equiations) The weight can be measured on the scale with the given weights, supposing there are 3 1kg weight, 2 3kg weight in total.

Sol. $f(x)=(1+x)^3(1+x^3)^2=1+3(x+x^2+x^3)+6(x^4+x^5)+3(x^6+x^7+x^8)+x^9$, the index points out the scale of each weight, the item 1 and x implies whether to use the weight, and the final expression means that there are 3 methods to measure those weighs 1-3 and 6-8kg, 1 for the 1kg and 9kg and 6 for the 4kg and 5kg(weights with the same scale is taken identical).

  • If allowed to put the object and weights on the same side, then $f(x)=(1+x+x^{-1})^3(1+x^3+x^{-3})^2$
  • If not to distinguish weights with the same scale, then$f(x)=(1+x+x^2+x^3)(1+x^3+x^6)$(only coefficients vary)

Problem B Number Division To divide m into the sum of many numbers.

  • If it is unordered and divided into n parts, $f(x)=(1+x+x^2+…+x^n)^m$, the coefficient of $x^m$ is thus obtained.
  • If it is ordered, then $f(x)=(1+x+x^2+…x^n)(1+x^2+…+x^{2\left \lfloor \frac{n}{2} \right \rfloor})…(1+x^{n-1})$
  • (Ferrers Diagram) If $N=\sum_{i=1}^n{x_ia_i}, l_i\leq x_i\leq u_i$ with $a_i$ given, then $f(x)=(x^{l_1a_1}+…x^{r_1a_1})…$

Theo. A decimal number and a corresponding binary number are mapped uniquely.

pf. For any integer, it can be decompleted into the sum of several items in $2^n$, and thus $f(x)=(1+x)(1+x^2)(1+x^4)…=\frac{1-y^2}{1-y}·\frac{1-y^4}{1-y^2}…=\frac{1}{1-y}=\sum y^i$, the coefficient 1 determines an unique expression in existence.

Notes. Expential Generating Function, widely applied in sequences to do with combination numbers, is defined as $g(x)=1+x+{x^2\over2!}+…=e^x, g(x)=1-x+{x^2\over2!}+..=e^{-x}$

Problem C(Arrangement from multisets) If there are $n_i$ pieces of object k, how many arrangements are there to pick out m pieces?

Analysis If m=2, then we have $m!\sum \frac{1}{n_i!n_j!}$. To construct a formula in such shape, we turn to exponential function, whose coefficients reveal the fact. $$G(x)=(1+\frac{x}{1!}+…+\frac{x^n_1}{n_1!})…(1+\frac{x}{1!}+…+\frac{x^n_k}{n_k!})\tag{3.1}$$

Problem C(2) Use 4 colors to dye n blocks and the amount of the red and green are even. How many arrangements are there?

  Sol. $f(x)=(1+\frac{x^2}{2!}+..)^2(1+\frac{x}{1!}+…)^2=(\frac{e^x+e^{-x}}{2})^2e^{2x}$

3.2 Chessboard Polynomials

In the particular problem 8 queens, it wonders the amount of the layout such that no queens lie in the same column, row or diagnal. And now there is no restriction on diagonals.

Write the chessboard as C and the queen amount $k$, and $n_k(C)$ is to be calculated. If C can be divided into two parts, $C_b$ cast away one block in C and $C_r$ cast away the column and row the block lie in. Then we have: $$n_k(C)=n_{k-1}(C_r)+n_k(C_b), n_k(C)=\sum_{j=0}^k n_j(C_r)n_{k-j}(C_b)\tag{3.2}$$

Based on (3.2), the generating functions are:($C_1,C_2$ are discrete) $$f(C)=xf(C_r)+f(C_b), f(C)=f(C_1)f(C_2)\tag{3.3}$$

Ifthe area of forbidden zone in a rectangle-grid are in the minority, it is wise to consider from the opposite. According to exclusive-inclusive principle, we have:(C’ is the forbidden zone) $$n!n!-n_1(C’)(n-1)!(n-1)!+n_2(C’)(n-2)!(n-2)!-…\tag{3.4}$$

4 Stiring Numbers

4.1 First Kind Stiring Number

Scenario Supposing there are n people distributed into m loops, the amount of possible situations is defined as the first kind Stiring number $S(n,m)$.

Sol. Suppose to take in a newcomer, he can foster a loop on his own or find one to insert in where he can be on the left side of the n people already existing, then we have $$S(n,m)=S(n-1,m-1)+nS(n-1,m)\tag{4.1}$$ $$S(n,0)=0,S(n,1)=(n-1)!,S(n,n)=1 \tag{4.2}$$

Generating Function The rising factorial is the solution to (4.1) while the falling factorial meets another definition(4.5) $$x^{\bar n}=x(x+1)..(x+n-1),x^\underline{n}=x(x-1)…(x-n+1) \tag{4.4}$$ $$S(n+1,m)=S(n,m-1)-nS(n,m)\tag{4.5}$$

pf. Now focus on $f_n(x)=x(x+1)…(x+n-1)=\sum^{n-1} S(n,k)x^k,f_{n+1}=x(x+1)…(x+n)$ $=\sum^n S(n+1,k)x^k$, then for the term $x^k$, $S(n+1,k)$ partly derives from $f_n$ where n in the addictive term $(x+n)$ contributes to $nS(n,k)$; the rest comes from the x in $(x+n)$ which consists of $S(n-1,k-1)x^{k-1}·x$ for $x^k$. Now that for any k it holds water, so does m. □

Applications Given n buildings arranged in a row with different heights ranging from 1 to n, if a and b buildings can be seen from the left and right sides respectively, you are required to determine the possible number of arrangements for these buildings.

   Analysis The highest building serves as the ridge in the problem which can be seen from both sides. And on either side of it, there are a and b loops–the heights differ from one another making it only one possible arrangement to ensure only one visible building in a loop. Thus $\sum{C_n^iS(i,a)+C_n^{n-1-i}S(i,b)}(i\geq a, n-1-i\geq b)$ is obtained.

To get the table of S(n,m), we can choose to iterate by (4.1),(4.5) or adopt polynomial multiplication by FFT(O($n\log^2 n$)). To get the table of S(n,m), we can choose to iterate by (4.1),(4.5) or adopt polynomial multiplication by FFT(O($n\log^2 n$)).

4.2 Second Kind Stiring Number

Scenario Supposing there are n people distributed into m sets, the amount of possible situations is defined as the second Stiring number $S_2(n,m)$(which is abbreviated as $S(n,m)$)

Sol. Likewise, suppose n people set up within m rooms, then the newcomer can find a new room in or choose an exisiting one(rooms doesn’t differ). And thus obtaining $$S(n,m)=S(n-1,m-1)+mS(n-1,m)\tag{4.6}$$ $$S(n,0)=S(0,n)=0, S(n,n)=S(n,1)=1 \tag{4.7}$$

Given that the generating functions generally are 1D, we get m fixed and have $a_n=m!S(n,m)$(m! means are taken into account to adopt exponential generating function which implies ordered premise). Because each room is not empty, $g(x)=(e^x-1)^m=\sum_{i=0}^mC_m^ke^{kx}(-1)^{m-k}$,expand $e^{kx}$, $$g_m(x)=\sum_0^m(1+\frac{k}{1!}x+\frac{k^2}{2!}x^2…)C_m^k(-1)^{m-k} \tag{4.8}$$

Thus $a_n=m!S(n,m)=\sum_0^m(-1)^{m-k}C_m^kk^n$, i.e. $$S(n,m)=\frac{1}{m!}\sum_0^m(-1)^{m-k}C_m^kk^n\tag{4.9}$$

Applications Several placing-balls models(n balls differ).

  • m unordered non-empty boxes. S(n,m)
  • m ordered non-empty boxes. m!S(n,m)
  • m unordered boxes. $\sum_0^m S(n,i)$
  • m ordered boxes. $A_m^i\sum_0^m S(n,i)$

Notes. Two kinds of Stiring Numbers have a somewhat symetric property. $\Sigma_{k=0}^n S_1(n,k)S_2(k,m)=\Sigma_{k=0}^n S_2(n,k)S_1(k,m)$
